# Peak AI-kansenscan landingpage

Statische landingspage voor Meta-verkeer naar de gratis AI-kansenscan van Peak Automations.

## Lokaal bekijken

Open `index.html` rechtstreeks in je browser, of start een simpele lokale server:

```sh
python3 -m http.server 8000
```

Daarna staat de pagina op `http://127.0.0.1:8000`.

## Aanpassen

- Vervang de Maxim placeholder in `index.html`.
- De Calendly embed staat in de sectie `#plan`.
- Styling staat in `styles.css`.
